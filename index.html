Principais coisas que aprendi nesse tempo

HTML

É a linguagem padrão utilizada na construção de páginas web. Desenvolvida para estruturar o conteúdo na internet, o HTML permite a criação de documentos com texto, imagens, links, formulários e outros elementos interativos.

CSS

É utilizado para editar a apresentação e o estilo de um documento HTML. Com ele podemos definir como os elementos HTML devem ser exibidos em termos de layout, cores, fontes, e outras características visuais. Graças a ele nossos sites ficam bonitos.

Python

É uma linguagem de programação de alto nível, conhecida por sua simplicidade e legibilidade. Sua sintaxe clara e concisa facilita o aprendizado e a escrita de código eficiente, tornando-a uma escolha popular para os programadores.

GitHub

É uma plataforma onde além de poder guardar nossos códigos, nós também podemos ver os projetos feitos por outras pessoas. Ele foi muito importante principalmente para salvarmos nosso código para editar outro dia, ou para compartilhar ele com um colega.
